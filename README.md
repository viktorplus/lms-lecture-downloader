# LMS Lecture Downloader

Скрипт автоматически скачивает лекции с Moodle LMS (itcareerhub.de) и сохраняет их по предметам в указанную папку.

---

## Требования

- [Node.js](https://nodejs.org/) версии 18 или новее
- Windows 10/11

Проверить установку Node.js:
```
node --version
```

---

## Установка (один раз)

1. Открой терминал и перейди в папку скрипта:
   ```
   cd D:\dev\small_script\get_pdf
   ```

2. Установи зависимости:
   ```
   npm install
   ```

3. Установи браузер Chromium для Playwright:
   ```
   npx playwright install chromium
   ```

---

## Настройка

Открой файл `download_missing_subjects.js` в любом редакторе.

В самом начале файла найди блок **КОНФИГУРАЦИЯ** и измени нужные значения:

| Переменная    | Описание                                          | Пример значения                          |
|---------------|---------------------------------------------------|------------------------------------------|
| `TARGET_ROOT` | Папка, куда сохраняются лекции                    | `'D:/git/Archives/ich_pdf_lesson'`       |
| `LMS_BASE`    | Адрес LMS (менять только если сменился домен)     | `'https://lms.itcareerhub.de'`           |
| `PROFILE_DIR` | Папка с сохранённой сессией браузера              | по умолчанию рядом со скриптом           |
| `HEADLESS`    | `false` — браузер видимый, `true` — скрытый       | `false`                                  |

## Запуск

```
cd D:\dev\small_script\get_pdf
node download_missing_subjects.js
```

### Первый запуск

При первом запуске откроется окно браузера Chromium с сайтом LMS.

1. Войди в свой аккаунт вручную в этом браузере.
2. Убедись, что открылась страница **"Мои курсы"**.
3. Вернись в терминал и нажми **Enter**.

Сессия сохраняется — при следующих запусках логин не потребуется.

### Повторные запуски

Просто запусти команду снова. Скрипт:
- пропустит предметы, в которых уже есть файлы;
- найдёт новые курсы на LMS, которых ещё нет локально;
- создаст для них папки и скачает все лекции.

## Структура сохранённых файлов

```
TARGET_ROOT/
├── Python Advanced_Morning_2025_PD/
│   ├── 01 - Django. Введение/
│   │   ├── Django_1.pdf
│   │   └── Django_1 Конспект.pdf
│   ├── 02 - Django. Модели/
│   │   └── Django_2.pdf
│   └── ...
├── Docker & Cloud Tech_Morning_2025_PD/
│   └── ...
└── ...
```

- Каждый **курс** → отдельная папка с именем предмета.
- Каждая **секция** курса → подпапка вида `NN - Название секции`.
- Файлы сохраняются с оригинальными именами.
- Если файл с таким именем уже есть — к имени добавляется `(1)`, `(2)` и т.д.

## Возможные ситуации

| Ситуация | Что произойдёт |
|---|---|
| Новый предмет появился на LMS | Скрипт сам создаст папку и скачает лекции |
| Папка предмета уже есть и не пустая | Скрипт пропустит её |
| Курс на LMS без файловых ресурсов | В терминале появится сообщение `(нет файловых ресурсов)` |
| Сессия истекла | Откроется браузер — войди снова и нажми Enter |

## Вспомогательный скрипт: fix_empty_sections.js

Используется если нужно перезакачать конкретный предмет или отдельные секции в нём (например, после сбоя).

1. Открой `fix_empty_sections.js`.
2. В переменной `SUBJECTS_TO_RESCAN` укажи название папки предмета:
   ```js
   const SUBJECTS_TO_RESCAN = [
     'Python for AI_Morning_2025_PD',
   ];
   ```
3. Запусти:
   ```
   node fix_empty_sections.js
   ```

Скрипт пропустит уже заполненные секции и докачает только пустые.
